Coloca aquí las fotos reales de cada perfume (jpg/png/webp).

Luego, en src/data/products.js, en el campo `image` de cada producto,
escribe la ruta con el mismo nombre de archivo, por ejemplo:

  image: "/images/products/khamrah.jpg"

Si prefieres usar una URL externa (ej. un link directo a una imagen),
también funciona: solo pega la URL completa en `image`.
